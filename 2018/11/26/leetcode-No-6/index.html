leetcode 6、Z字形变换

发表于 2018-11-26

将一个给定字符串根据给定的行数，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 “LEETCODEISHIRING” 行数为 3 时，排列如下：

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如：”LCIRETOESIIGEDHN”。

请你实现这个将字符串进行指定行数变换的函数：
string convert(string s, int numRows);

示例 1:

输入: s = “LEETCODEISHIRING”, numRows = 3

输出: “LCIRETOESIIGEDHN”

示例 2:

输入: s = “LEETCODEISHIRING”, numRows = 4

输出: “LDREOEIIECIHNTSG”

解释:

个人见解：

解法一：

拿到该题时，最容易想到的思路应该就是直接把字符串转变成字符类型的二维数组，存入数组时按列存入，展示结果时，按行输出即可。

如题目描述，二维数组的行数即为numRows，而列数为numRows的情况下字符串s长度的最大情况。

以numRows = 3为例：

将这样的组合看作一个整体，字符串s的长度有多少个这样长度的总和，再算上这个组合所占用的列数，即为二维数组的总列数。

这样的组合所包含的字符数为 第一列为numRows数，而剩下的几列为numRows所占的字符数减去首尾两行的字符数

所以这样的组合的全部字符为 numRows + （numRows - 2） = 2 * numRows - 2

字符串所包含的这样的组合数为 s.length / (2 * numRows - 2)，整数除法运算是向下取整，所以结果需要再 + 1 保证列数的需要

当然直接 + 1会造成冗余，最大的冗余情况是当s.length / (2 * numRows - 2)能够整除时，此时会多计算出一个组合数，本例中允许这种冗余情况

这样的组合所占用的列数为 numRows 去掉首尾两行的字符数 再 加上第一列 即为 组合占用的列数 numRows - 2 + 1 = numRows + 1

所以最大列数为 (s.length / (2 numRows - 2) + 1) (numRows + 1)

public String convert(String s, int numRows) {
    if(numRows == 1){
        return s;
    }
    int numCols = (s.length() / (2 * numRows -2) + 1) * (numRows - 1);
    char[][] array = new char[numRows][numCols];
    /**
     * 初始化数组
     */
    for(int i=0; i< numRows ;i++){
        for(int j=0;j<numCols;j++)
            array[i][j] = ' ';
    }
    char[] stringToChar = s.toCharArray();
    /**
     * 用zCol来标明Z字形排列的当前列数，zCol = 1时,行数全部占满
     */
    int zCol = 1;
    /**
     * 存入时，外层循环控制列，内存循环控制行，其中n控制字符串s的每一位
     */
    for(int i = 0,n = 0;i < numCols; i++){
        if(n >= stringToChar.length) {
            break;
        }
        for(int j=0;j<numRows;j++){
            /**
             * 若zCol为1，则每一个行都存入值
             * 否则，只有numRows - zCole存入值
             */
           if(zCol == 1){
               array[j][i] = stringToChar[n++];
           }
           else {
               if(j == numRows - zCol)
                   array[j][i] = stringToChar[n++];
           }
           if (n >= stringToChar.length) {
               break;
           }
        }
        /**
         * 若zCol的下一列为numRows，则置zCol为1，重新开始计算
         * 否则，zCol+1
         */
        zCol = (zCol + 1) == numRows ? 1 : zCol + 1;
    }

    String result = "";
    for(int i = 0;i < numRows;i++){
        for(int j=0;j<numCols;j++){
            if(array[i][j] == ' ')
                continue;
            result += array[i][j];
        }
    }
    return result;
}

代码中用zCol来表示Z字形的当前的列数，zCol以上述提到的一个组合为一个循环周期。

以numRows = 4的组合为例，最左边的数字表示行下标，第一列是满行的，此时zCol为1，
下一列zCol为2，此时只有行下标为2时有值，zCol为3时，只有行下标为1有值，所有在zCol不为1的列，行下标满足 numRows - zCol的关系。

当zCol+1 == numRows时，表示当前组合结束，进入下一组合，此时置zCol = 1，重新开始计数。

解法二：

由解法一可以进一步联想到，解法一是用二维数组来存放Z字形字符，如果直接用下标来控制字符的输出，可以直接来模拟二维数组的效果。


考虑numRows为4的情况，第一行和最后一行最容易判断，直接算间隔就行，即 2 * numRows - 2

第二行时，E到O的间隔，其实相当于是numRows为3时，第一行的间隔，即 2 * 3 - 2

所以在中间行的间隔是 2 * （numRows - row) - 2 ,其中row是当前行，从0开始

那再计算O到第二个E的距离时，应该是numRows = 4时的间隔减去numRows = 3时的间隔，即2 (numRows) - 2 - [2 (numRows - row) -2] = 2 * row

以此类推，直到最后一行

public String convert(String s,int numRows){
    if(numRows <= 1){
        return s;
    }
    List<StringBuilder> stringList = new ArrayList<>();
    for(int i = 0;i < numRows;i++){
        stringList.add(new StringBuilder());
    }
    int n = 0,row = 0;
    /**
     * isMiddle 用来标识是否是非首尾行的中间项
     */
    boolean isMiddle = false;
    while(n < s.length() && row < numRows){
        if(row == 0 || row == numRows - 1){
            int jump = 2 * numRows - 2;
            stringList.get(row).append(s.charAt(n));
            n += jump;
        }
        else {
            int jump;
            if(!isMiddle){
                jump = 2 * (numRows - row) - 2;
                isMiddle = true;
            }
            else {
                jump = 2 * row;
                isMiddle = false;
            }
            stringList.get(row).append(s.charAt(n));
            n += jump;
        }
        if(n >= s.length()){
            isMiddle = false;
            row++;
            n = row;
        }
    }
    StringBuilder stringBuilder = new StringBuilder();
    for(StringBuilder tmp : stringList){
        stringBuilder.append(tmp);
    }
    return stringBuilder.toString();
}

官方题解：

解法一： 按行排序

上面的解法拘泥于去模仿Z字形的原状，其实可以更简单化。

这种形式，其实可以当成打字机，使用竖条状折叠的打纸带，一次打印一个字符，打一个移动一下，当打纸带用完后，再从底向上打一遍，如此反复。

按行排序这种解法就是这种思路。

public String convert(String s, int numRows) {

    if (numRows == 1) return s;

    List<StringBuilder> rows = new ArrayList<>();
    /*
        先初始化行数，有可能会出现numRows大于s.length的情况
        类似于打纸带长度大于要打的字符数，只需打一条即可
        所以这里使用Math.min(numRows, s.length())作为结束条件
    */
    for (int i = 0; i < Math.min(numRows, s.length()); i++)
        rows.add(new StringBuilder());

    int curRow = 0;
    boolean goingDown = false;

    for (char c : s.toCharArray()) {
    /*
        按竖条状打印，但是rows记录的是将打纸带并列时的情况，所以直接在当前行追加当前字符，最后拼接就好
    */
        rows.get(curRow).append(c);、
    /*
        只有打印到第一行或者最后一行时才转向
    */
        if (curRow == 0 || curRow == numRows - 1) goingDown = !goingDown;
        curRow += goingDown ? 1 : -1;
    }

    StringBuilder ret = new StringBuilder();
    for (StringBuilder row : rows) ret.append(row);
    return ret.toString();
}

解法二：按行访问

这种解法和上述解法二的思路相同，即模拟二维数组的存储情况，但相比上述解法二，其总结出了更普适性的计算规律

public String convert(String s, int numRows) {

    if (numRows == 1) return s;

    StringBuilder ret = new StringBuilder();
    int n = s.length();
    int cycleLen = 2 * numRows - 2;

    for (int i = 0; i < numRows; i++) {
        for (int j = 0; j + i < n; j += cycleLen) {
            ret.append(s.charAt(j + i));
            if (i != 0 && i != numRows - 1 && j + cycleLen - i < n)
                ret.append(s.charAt(j + cycleLen - i));
        }
    }
    return ret.toString();
}
